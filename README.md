# BankApi

### To start project
0. Prerequisites:
    - Have Java 17 or later installed
    - Have Maven 3.5+ installed
1. Navigate to the project folder: 
    - On Windows: `cd ..\BankApi`
    - On Linux: `cd BankApi`
2. Run the project: 
    - On Windows: `./mvnw spring-boot:run`
    - On Linux: `mvn spring-boot:run
